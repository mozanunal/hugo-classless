Section 3: Code, Quotes, and Images

Inline code is simple: const a = 1;. But a full code block with syntax highlighting is more interesting.

# A simple Python script
def fibonacci(n):
    a, b = 0, 1
    while a < n:
        print(a, end=' ')
        a, b = b, a+b
    print()
